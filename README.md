## Settings

- **connect-gdrive** you need to make a alias link to **japl-android-google-connect-module** 
> ln -s ~/git/japl-android-google-connect-module ~/[AndroidWorkSpace or Git]/[CR Alameda 181 App]/connect-gdrive
